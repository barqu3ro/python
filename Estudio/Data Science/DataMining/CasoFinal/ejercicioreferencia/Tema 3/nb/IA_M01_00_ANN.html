Problema de regresin lineal para obtener los grados Fahrenheit (y) en funcin de los Celsius (x)

Entendimiento del problema

Queremos obtener una primera red neuronal de una neurona con funcin de activacin lineal para que aprenda la siguiente funcin:

$$ f = c \times 1.8 + 32 $$

Aunque sabemos como se convierte queremos que la red lo aprenda sola a partir de solo unas observaciones

Seleccin modelo

Una red neuronal con 1 entrada (c) y funcin de activacin lineal:

In[6]:
###############################################
# MLP con 1 capa
###############################################

# La definicin del modelo toma una lista de capas como parmetro
# que especifica el orden de clculo desde la entrada a la salida:
# - input_shape=[1]  Especifica que la entrada a la capa tiene un valor simple (x1)
# - units=1  Nmero de neuronas en la capa.

capa = tf.keras.layers.Dense(units=1,input_shape=[1])
model = tf.keras.Sequential([capa])

Construccin modelo

In[7]:
# Se compila el modelo con dos parmetros:
# - Loss function  Una forma de medir lo lejos que estn las predicciones del resultado deseado.
#                 (La diferencia se llama prdida o "loss".)
# - Optimizer function  Una forma de ajustar los valores internos para reducir la prdida con tasa aprendizaje

model.compile(optimizer=tf.keras.optimizers.Adam(0.1), loss="mean_squared_error")
